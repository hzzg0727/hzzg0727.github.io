    Installation

    Configuration:
Main system: Windows 11

    启用 Windows 功能

    运行启用或关闭 Windows 功能，开启以下功能：

    • Windows 虚拟机监控程序平台
    • 适用于 Linux 的 Windows 子系统
    • 虚拟机平台

    注：若未执行此步骤，直接安装 WSL 会有请求超时的报错。

    安装 WSL

    Powershell 运行如下命令

    wsl --install

    如果没有默认安装 Ubuntu，则按照提示使用如下命令选择想要的 Linux 发行版即可。

    wsl --install -d <Distribution Name>

    设置 WSL 开发环境

    参考 WSL 设置开发环境的官方教程。

    Installing MEEP on Linux Subsystem

    在 Windows 本地安装 MEEP 并不被支持。采用 WSL 的 Linux 子系统安装和使用 MEEP 是一个很实用和便捷的选择。

    参考 Linux 子系统安装 MEEP 的英文教程 和 Linux 子系统安装 MEEP 的中文教程。

    实用 Linux 指令

    shell 重启

    运行如下命令，会在当前进程中启动一个新的 bash shell。 执行该命令后，当前 shell 会被替换为新的 bash shell，相当于重新启动了 shell [1]。

    exec bash
